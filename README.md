# Python for Data Analysis Continuous Internal Assessment 3
Repository for hosting Python CIA 3 code and other things
Team members: Kevin Joseph and Jiya Bansal
We have created an Insurance Policy Management System that records the insurance policies issued to customers.It has a menu drive interface that allows insurance administrators to manage customers and insurance policies effectively.
Objective: To build a modular, class-based system for managing insurance policy holders records  with integrated error handling.
Error Handling: Implemented try-except blocks to handle ValueError (invalid age inputs), KeyError (searching non-existent IDs), and RuntimeError (logic violations).
